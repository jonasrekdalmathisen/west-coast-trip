# 🗺️ West Coast Road Trip — Jun 14 – Jul 14

Interactive planning map for a 4-week road trip from San Diego to Seattle along Hwy 1 and Hwy 101.

**[→ Open the map](https://jonasrekdalmathisen.github.io/west-coast-trip/)**

## Map layers

Toggle any layer on/off using the control panel on the right side of the map.

| Layer | What's on it |
|-------|-------------|
| **Main Stops** | The 8 overnight bases — click for dates, highlights, drive time to next stop |
| **Route** | Drive route along Hwy 1 / Hwy 101 — hover for drive times |
| **Day Trips** | Confirmed day trips from each base (amber) |
| **Maybe / Optional** | Worth considering but not locked in (yellow) |
| **National Parks** | 10 NPs near the route, colour-coded by how reachable they are |
| **Hikes** | 22 hikes, colour-coded by difficulty — with distances, elevation, permit info |
| **Snorkel / Dive** | 9 snorkel & freedive sites with water temp, gear and operator info |
| **Bike Routes** | 17 routes colour-coded by surface (paved / gravel / mixed) |
| **Bike Rentals** | 12 rental shops along the route |
| **Whale Zone** | Santa Barbara Channel (blue whales, Jun–Jul) + Depoe Bay (grey whales) |

### ⭐ Add to plan
Every popup has an **Add to plan** button. Saved items appear in the **My Plan** sidebar (top right) and persist across sessions via localStorage.
